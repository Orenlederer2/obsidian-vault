# פרויקטים של SEOEasy

## 🚀 בעדיפות גבוהה

### תוכן ודפים נוספים
- [ ] להתייעץ עם Claude Code על מילות מפתח מהבדיקה
- [ ] לבנות 2-3 דפים חדשים לפי ממצאי הבדיקה

### עדכוני דף הבית
- [ ] להראות מה כלול בחבילה (כמה מילות מפתח למעקב)
- [ ] לשפר את המאמר בדף הבית
- [ ] שינוי מחירים: ₪800/חודש, ₪1,500 הקמה
- [ ] להוסיף כרטיס נוסף: בניית אתר ב-₪1,500

## 🔧 פיתוח טכני - פורטל לקוחות

- [ ] מעבר אוטומטי לפורטל אחרי תשלום
- [ ] אינטגרציה עם אפליקציית מעקב מילות מפתח
- [ ] אפשרות שדרוג: בניית אתר ב-₪1,000 (חד פעמי)

## 💰 שירותים נוספים להכנסה

### תוכן AI לעסקים (₪1,000/חודש)
- יעד: 8 לקוחות
- שירות: 8-10 פוסטים לחודש
- קהל יעד: בעלי עסקים שלא יודעים ChatGPT
- מצב: טרם התחלתי

### בניית אתרים מהירה (₪1,500-2,000)
- שיווק: "אתר תוך 48 שעות"
- נישות: מספרות, קליניקות, עסקים מקומיים
- מצב: טרם התחלתי
